# VANTA FITNESS

Premium cinematic gym website built locally with Vite, React, Tailwind CSS, React Router, Framer Motion, and Lucide React.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Notes

- No deployment configuration is included.
- No backend, auth, or database is included.
- Placeholder imagery is centralized in `src/data/gymData.js`.
